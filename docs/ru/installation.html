Установка

Замечание: для чтобы настроить редактор для использования с React + BuckleScript см. здесь.

BuckleScript

BuckleScript компилирует ReasonML код в JavaScript. Вы можете установить его с помощью:

npm install --global bs-platform
bsb -init my-react-app -theme react-hooks
cd my-react-app && npm install && npm start
# в другой вкладке
npm run webpack

Система сборки BuckleScript'а bsb имеет команду init которая создает шаблон проекта. Тема react предостовляет легкое решение во избежание сложного обучения и для простоты интеграции в существующий проект.

Система сборки компилирует в обычые JS файлы, так что вы можете открыть index.html прямо из файловой системы. Без необходимости в локальном сервере.

Добавление Reason + Bucklescript в существующий проект

Установите зависимости:

yarn add bs-platform --dev --exact
yarn add reason-react --exact

Добавьте скрипты в файле package.json:

"scripts": {
  "re:build": "bsb -make-world -clean-world",
  "re:watch": "bsb -make-world -clean-world -w"
}

Создайте файл bsconfig.json в корневой папке вашего проекта и добавьте следующее содержимое. Поле name можете изменить.

{
  "name": "your-project-name",
  "reason": { "react-jsx": 3 },
  "bsc-flags": ["-bs-super-errors"],
  "sources": [
    {
      "dir": "src",
      "subdirs": true
    }
  ],
  "package-specs": [
    {
      "module": "es6",
      "in-source": true
    }
  ],
  "suffix": ".bs.js",
  "namespace": true,
  "bs-dependencies": [
    "reason-react"
  ],
  "ppx-flags": [],
  "refmt": 3
}
