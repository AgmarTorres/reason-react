Exemple d’introduction

Avant de commencer, voici un petit aperçu de l’API ReasonReact. Pas de soucis si certains éléments ne vous sont pas familiers ; la documentation couvre l’ensemble d'entre eux.

Le composant "Salut"

/* fichier: Greeting.re */

[@react.component]
let make = (~name) =>
  <button> {ReasonReact.string("Hello " ++ name ++ "!")} </button>;

Using Greeting in your App

If you're writing your entire React app in Reason, you'll probably have a ReactDOM.render in an index.js. This is what that looks like in Reason:

/* file: Index.re */
ReactDOMRe.renderToElementWithId(<Greeting name="John" />, "root");

This is how you used to write this in plain Javascript (index.js):

/* file: index.js */
ReactDOM.render(<Greeting name="John"> document.getElementById("root"));

Using Greeting in an existing Javascript/Typescript App

It's easy to import a Reason component into your existing app. All Reason extensions will have bs and export your component as make. You can rename it and call it as you usually do. Gentype makes this easier (see the example).

/* file: App.js */

import { make as Greeting } from './Greeting.bs'

export default function App() {
    return (
        <div>
            <Greeting name="Peter" />
        </div>
    )
}

Make sure you check out Examples for more!
